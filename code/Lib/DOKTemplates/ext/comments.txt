Для диплома - один из моментов был - статическое копирование функций.
Т.е. когда берём одно пространство имён и просто копируем его в другое.

Минус этого подхода - нельзя потом переопределять извне.

Интересный момент - вложенность вызовов (проблема рекурсии).
x
 x.y
  x.y.z
   x.y
   
x
 .y
  .z
   x.y <- в простом случае не разрулится
   
решение - уровни вызова
x
 0.y
  0.y.z
   0.y  
   
x
 0.y
  1.z
   0.y
